# WORDLY
This is a Summative Lab for Module 2.

- Wordly is a Single Page Application that allows users to search for words, get their details and save them to favourites.

- Words saved to favourites can also be deleted when needed.

### Example of word details are:
    1. Phonetics (text and audio)
    2. Part of Speech
    3. Definition 
    4. Example.

### Main Objectives

- The purpose of this Summative Lab was to demonstrate the use of fetching resources from the API:`https://dictionaryapi.dev/`.
